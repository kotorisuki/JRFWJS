\subsection{数据来源}
\begin{itemize}
\item 国泰安CSMAR数据库专注于学术高校研究

\item 覆盖经济、股票、基金、债券等领域，涵盖面广。

\item 数据做深层衍生加工，而CSMAR侧重于数据的深度，强调对原始数据的深层专业加工、整理，属精准数据。

\item CSMAR支持EXCEL、TXT、XML、 SAS 、STATA 、MATLAB 、R等多种格式。

\end{itemize}
