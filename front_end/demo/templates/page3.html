Batch Calculation
Upload a json file which incluing all the bond for query. For each bond the following information should be contained:
  • face_value: the face value of the bond
  • coupon_rate: the coupou rate of the bond
  • time_start: the issue date of the bond, format: YYYY-MM-DD
  • year: the length of the term of the bond
  • time_cur: selling date, the same format as time_start
  • frequency: the frequncy of the payment per year
